Setup Instructions

Accessing the App

  1. Open your browser and go to the preview/staging URL (provided by Jasper) or https://app.lobuddy.com for production
  2. Log in with your credentials
  3. You should see the main dashboard with the pipeline view

Before You Start Testing

  1. Use Chrome or Edge (recommended browsers)
  2. Make sure your microphone is enabled for voice testing (browser will ask for permission)
  3. Have this guide open in a separate tab for reference
  4. Keep a copy of the Reporting Template (at the bottom) open to record results

Module 1: Capture

  2. Say or type the sample prompt below
  3. Wait for the system to process

Module 2: Follow-Up Coach

Task Creation

Creates follow-up tasks for the right user on the right opportunity.

Test Steps

  1. Navigate to an existing opportunity (use the one created in Test 1, or any existing one)
  2. Via voice or chat, say the sample prompt below
  3. Check the Tasks section on the opportunity detail page
üé§ Say or Type
"Remind me to follow up with John Smith in 2 days"
‚úì Expected ‚Äî Pass
  • A new task appears with due date 2 days from now
  • Task is assigned to you (logged-in LO)
  • Task is linked to the correct opportunity
  • Task description references the follow-up
‚úó Failure Indicators
  • No task created

  • Task on the wrong opportunity

Module 3: Qualify

Discovery Call Data Capture

Captures borrower qualification data from a conversation and updates the opportunity.

Test Steps

  1. Open an opportunity that's in "Contacted" or "Pre-Qualified" status
  2. Via voice or chat, provide the qualification info below
  3. Check the opportunity detail page for updated financial data
üé§ Say or Type
"John Smith makes $85,000 a year, has about $15,000 in savings, his credit score is 720, and he has $500 in monthly debt payments. He's been at his job for 3 years."
‚úì Expected ‚Äî Pass

‚úó Failure Indicators

Module 4: Validator

Status Transition Checks

Checks that required data is present before allowing a status change.

Test Steps

  1. Find or create an opportunity in "Pre-Qualified" status
  2. Try to advance it to "Pre-Approved" by clicking the status change button
  3. The system should show what requirements are met and what's missing
‚úì Expected ‚Äî Pass
  • Validation summary appears showing required items
  • Missing items clearly listed (e.g., "Credit Report needed")
  • Can still proceed (warns but doesn't hard-block)
  • Completion percentage is shown
‚úó Failure Indicators
  • No validation ‚Äî status just changes
  • Validation shows incorrect requirements
  • System crashes or shows an error
  • Status change completely blocked, no way to proceed

Module 5: Realtor

Partner Portal

Provides realtors with a filtered view of their referred deals.

Test Steps

  1. Log out of your LO account
  2. Log in as a realtor/partner account (credentials provided separately)
  3. Check what you can see on the dashboard
‚úì Expected ‚Äî Pass
  • Only opportunities referred by this realtor visible
  • Borrower contact info visible (name, phone, email)
  • Internal notes and credit scores NOT visible
  • Can view loan status and scenarios
  • Can request a pre-approval letter
  • Cannot change loan status or edit financials
‚úó Failure Indicators
  • Realtor can see ALL opportunities
  • Realtor can see internal notes or credit scores
  • Realtor can perform unauthorized actions
  • Empty dashboard when referrals should exist
  • Error on login or page load
