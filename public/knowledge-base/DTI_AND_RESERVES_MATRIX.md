# DTI Limits & Reserve Requirements by Program

## DTI Limits

| Program | AUS Approval | Manual Underwrite | Notes |
|---|---|---|---|
| **Conventional (Fannie)** | Per DU (typically ≤ 50%) | 36/45 (50 w/ comp factors) | DU can approve higher with strong file |
| **Conventional (Freddie)** | Per LP (typically ≤ 50%) | 33/43 (45 w/ comp factors) | Slightly more conservative than Fannie manual |
| **FHA** | Per TOTAL (up to 56.99%) | 31/43 (37/47 w/ comp factors) | See FHA manual matrix for score tiers |
| **VA** | Per AUS (no hard cap) | 41% back-end guideline | Residual income test is the real gatekeeper |
| **USDA** | Per GUS (29/41 typical) | 29/41 | Very strict on manual |
| **Non-QM/DSCR** | N/A — no DTI calc | N/A | Qualify on DSCR or bank statements |
| **UWM HELOC (Pink)** | Per guidelines | Index + Margin + 2% qual rate | See UWM Pink guidelines |

## Reserve Requirements

### Conventional (Fannie Mae)

| Scenario | Minimum Reserves |
|---|---|
| 1-unit Primary | Per DU (often 0 months) |
| 2-unit Primary | 2 months PITIA |
| 3-4 unit Primary | 6 months PITIA |
| Second Home | 2 months PITIA |
| Investment (1-4 unit) | 6 months PITIA |
| Multiple financed properties (5-10) | 6 months subject + 2% UPB of all other financed |
| High-balance loans | Per DU |
| Manual underwrite | 2 months (comp factor: 6+ months) |

### FHA

| Scenario | Minimum Reserves |
|---|---|
| 1-2 unit | Per TOTAL (often 0-1 month) |
| 3-4 unit | 3 months PITIA |
| Manual underwrite | 1-2 months (comp factor: 3+ months) |

### VA

| Scenario | Minimum Reserves |
|---|---|
| 1-4 unit | Per AUS (residual income is primary test) |
| Manual underwrite | No hard requirement but helps |

### Non-QM / DSCR

| Scenario | Typical Reserves |
|---|---|
| DSCR ≥ 1.25 | 3-6 months PITIA |
| DSCR 1.0-1.24 | 6-12 months PITIA |
| DSCR < 1.0 | 12+ months PITIA (if allowed) |
| Bank statement loans | 6-12 months typical |

## What Counts as Reserves

| Asset Type | Conventional | FHA | VA |
|---|---|---|---|
| Checking/Savings | 100% | 100% | 100% |
| Stocks/Bonds/Mutual Funds | 70% | 70% | 70% |
| Retirement (vested) | 60% | 60% | 60% |
| Cash Value Life Insurance | Net surrender value | Net surrender value | Net surrender value |
| Gift Funds | ❌ No | ❌ No | ❌ No |
| Borrowed Funds | ❌ No | ❌ No | ❌ No |
| Sale Proceeds (subject) | ❌ No | ❌ No | ❌ No |
| Business Assets | ❌ No | ❌ No | ❌ No |
| Crypto | Per investor | ❌ No | ❌ No |
